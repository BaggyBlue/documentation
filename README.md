# BaggyBlue Documentation

This is the open source documentation hub for BaggyBlue.

## Contents
* **Guides:** Instructions for setup and usage.
* **Errors:** Detailed explanations for common issues.

## Usage
The documentation is contained entirely in `index.html`. To view it, clone the repository and open the file in any web browser.
